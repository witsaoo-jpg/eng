ğŸ”§ Configuration

The app uses Claude API for AI responses. The API calls are made directly from the browser with no backend required.

Note: The Claude API endpoint is accessible from claude.ai artifacts without requiring an API key. If you want to deploy this elsewhere, you'll need to:

    Get an API key from Anthropic
    Set up a backend proxy to secure your API key
    Update the fetch URL in the code
